## üéØ Instructions - How to set up this small app

#### In your terminal, run these commands (you only need to do this once):

```sh
# Step 1. Clone this repository from Github
git clone https://github.com/jesusoterogomez/tasks-app-demo

# Step 2. Change into the new directory
cd tasks-app-demo

# Step 3. Install the software that we need to run this server
npm install
```

### To run the app

Run this in your terminal while in the `tasks-app-demo` directory

```sh
# Step 4. Starts the server in the URL http://127.0.0.1:4000
npm start
```

### ‚úÖ That's it

You can view the application by going to the URL: http://127.0.0.1:4000 in your Web Browser, when the server is running.

To close the server, press this key combination on your terminal:
- ü™ü In Windows: Control + C
- üçé In Mac: Command + C
